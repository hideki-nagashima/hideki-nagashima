## Philosophy

AIは人を置き換えるものではなく、人がより高い価値を生み出すための基盤だと考えています。

そのため、AI開発ではモデルやツールの性能だけでなく、業務プロセス、権限設計、監査性、組織の意思決定まで含めて設計することが重要です。AIを「便利な機能」として追加するのではなく、企業が安全に使い続けられる仕組みとして実装することを重視しています。

I believe AI should augment human judgment, expand organizational capability, and create systems where people can focus on higher-value work.
